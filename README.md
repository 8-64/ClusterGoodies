# ClusterGoodies
Collection of scripts and automations that make working and living with Kubernetes cluster (especially AWS EKS) easier

---

## What is here

| Tool | Purpose |
|:-----|:-------:|
| **kcc** | Kubernetes Current Context - script to automate switching between multiple AWS EKS accounts and cluster. Just store it somewhere in the $PATH, use AWS credentials as environmental variables, and run `kcc` |
